# Task Management System (Kanban Based)

This is a simple full-stack Task Management application built as part of an SDE assignment.
The application allows users to manage tasks using a Kanban board.

## Features
- User registration and login
- JWT-based authentication
- Create, view, update, and delete tasks
- Kanban board with three columns:
  - Pending
  - In Progress
  - Completed
- Drag and drop tasks between columns
- Tasks are user-specific

## Tech Stack
- Backend: Node.js, Express.js
- Frontend: HTML, CSS, JavaScript
- Database: MongoDB
- Authentication: JWT

## Project Structure
- `server.js` – Backend server and APIs
- `index.html` – Frontend UI with Kanban board
- `.env` – Environment variables
